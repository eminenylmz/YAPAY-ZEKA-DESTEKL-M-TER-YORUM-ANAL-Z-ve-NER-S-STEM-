# Yapay Zeka Destekli Müşteri Yorum Analizi ve Öneri Sistemi

Bu proje, e-ticaret platformlarındaki müşteri yorumlarını analiz eden web tabanlı bir sistemdir.

## Teknolojiler
- C#
- ASP.NET Core Web API
- MSSQL
- HTML, CSS, JavaScript

## Açıklama
Sistem, müşteri yorumlarını olumlu, nötr ve olumsuz olarak sınıflandırır ve sonuçları grafiklerle sunar.

## Proje Dokümanı
Detaylı proje dokümanına yukarıdaki dosyadan ulaşabilirsiniz.
